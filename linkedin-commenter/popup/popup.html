LinkedIn Gemini Commenter

Generate professional LinkedIn comments using Google Gemini AI.

Get your API key from Google AI Studio

How to use:

  1. Enter your Gemini API key above and click Save
  2. Navigate to any LinkedIn post
  3. Click on a comment box
  4. Click "Generate Comment" to create an AI-powered response
  5. Use "Regenerate" to create a different comment

Your API key is stored locally and never shared.
